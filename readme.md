bark
====

bark is a simple command line bookmarking tool


Usage
-----

Add a bookmark:
```sh
bark add <URL>
```

List bookmarks:
```sh
bark list
```

Archive a bookmark:
```sh
bark archive <ID>
```

Permanently delete a bookmark:
```sh
bark delete <ID>
```

List archived bookmarks:
```sh
bark list archived
```
