bark
====

bark is a simple command line bookmarking tool


Usage
-----

Add a bookmark:
```sh
bark add <URL>
```

List bookmarks:
```sh
bark list
```

Remove/Archive a bookmark:
```sh
bark delete <ID>
```

Show archived bookmarks:
```sh
bark archive
```
