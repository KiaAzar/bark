# bark

bark is a simple command line bookmarking tool

## Usage

### Add a bookmark

```sh
bark add <URL>
```

### List bookmarks

```sh
bark list
```

### Open a bookmark

```sh
bark open <ID>
```

### Archive a bookmark

```sh
bark archive <ID>
```

### Permanently delete a bookmark

```sh
bark delete <ID>
```

### Get help

```sh
bark help
```

### Get help for a specific command

```sh
bark help <COMMAND>
```
